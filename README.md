# Lua-source-Optimizer
Lua source optimizer is a Source-to-Source Lua Optimizer,
it means that it gets a Lua code as input and outputs the Optimized version.

Game creators or Mod developers that use Lua as their Primary or Binding language may not Optimize their code
Making the game slow because of a badly-optimized Lua code,
This code is meant to Lex/Parse produce IR and Attempt to optimize every part of the Code
and then Recreating the Optimized lua source code,

## Current state
- This is a solo and slow project
- Super buggy and not support for all Lua syntaxs (for now)
- No Optimization passes (for now)
- Not very optimized
